Anomaly Detection in Network Traffic using Isolation Forest

This project applies the Isolation Forest algorithm to the KDD Cup 1999 dataset for detecting anomalies in network traffic data.

Included Files:
- Detailed PDF Report
- PowerPoint Presentation
- requirements.txt (Python dependencies)
- readme.txt (This file)

To run the project:
1. Install dependencies using `pip install -r requirements.txt`
2. Download the dataset from https://www.kaggle.com/datasets/galaxyh/kdd-cup-1999-data
3. Extract corrected.gz and place it in the working directory.
4. Run the Python script to analyze data.
